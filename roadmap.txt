На сайте:
1. делаем панель и сохранение связки url -> строка
2. вводим "тип страницы", для этого учимся сохранять привязки типа url pattern -> page type
3. добавляем блоки и связи с типами страниц (блок = css-selector)
5. делаем возможность выбора блоков из списка и автопоиск блоков на странице неизвестного типа (при создании нового)

Браузер связей:
1. делаем отображение сущностей с их связями в виде графа - изначально пустая страница, можно выбрать элемент из дерева сущностей сайта или текстовым поиском (ищем по названию)
2. делаем возможность создать виртуальные сущности (не представленные на сайте)
3. даем возможность добавить текстовое описание к любой сущности
4. делаем теги у сущностей
5. дорабатываем возможность выбора элемента для графа поиском по описаниям и тегам
